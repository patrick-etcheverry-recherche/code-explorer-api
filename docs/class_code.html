Description détaillée

Une classe pour représenter les composants d'un code source sous forme d'objets.

Documentation des constructeurs et destructeur

◆ Code()

Code::Code ( string  cheminFichier)

Constructeur : charge un code se trouvant au sein d'un fichier.

Le fichier contenant le code à charger est défini à partir de son nom système.
Le code chargé est analysé et décomposé en objets (Information, Traitement, Commentaire...) reliés entre eux (tel Commentaire est associé à telle Information, telle Information intervient dans tel Traitement...)

Paramètres
[in]cheminFichierLe chemin d'accès au fichier contenant le code source.
Par exemple : "/home/pierre/prog/helloWorld.php"

◆ getCommentaires()

Commentaire Code::getCommentaires ( string  typeCommentaire)

Identifie touts les commentaires utilisés dans le code source.

Les commentaires identifiés peuvent être de tout type : il peuvent porter sur des informations, sur des traitements ou sur le code dans sa globalité.

Paramètres
[in]typeCommentaireDésigne le type de commentaire à récupérer. Les valeurs possibles pour ce paramètre sont :
  • "all" pour récupérer tous les commentaires présents dans le code ;
  • "code" pour récupérer uniquement le commentaire global attaché en entête du code;
  • "information" pour récupérer uniquement les commentaires associés à des informations;
  • "traitement" pour récupérer uniquement les commentaires associés à des traitements.
Renvoie
Un tableau d'objets Commentaire représentant tous les commentaires insérés dans le code.
Les commentaires sont ordonnés selon leur ordre d'apparition dans le code. Le tableau retourné peut être vide si aucun commentaire n'est défini dans le code.

◆ getInformations()

Information* Code::getInformations ( string  typeInformation)

Identifie les informations utilisées dans le code source.

Les informations identifiées peuvent être de tout type : il peut s'agir
variables, de constantes, de compteurs, d'indice de boucles etc.

Paramètres
[in]typeInformationDésigne le type d'information à récupérer. Les valeurs possibles pour ce paramètre sont :
  • "all" pour récupérer toutes les informations ;
  • "const" pour récupérer uniquement les constantes;
  • "magic" pour récupérer uniquement les nombres magiques;
  • "var" pour récupérer uniquement les variables;
  • "simpleVar" pour récupérer uniquement les variables simples;
  • "composedVar" pour récupérer uniquement les variables structurées;
  • "accu" pour récupérer uniquement les variables de type accumulateur;
  • "count" pour récupérer uniquement les variables de type compteur;
  • "index" pour récupérer uniquement les variables de type indice de boucle.
Renvoie
Un tableau d'objets Information représentant toutes les informations utilisées dans le code ou uniquement celles d'un certain type.
Le tableau retourné peut être vide si aucune information du type demandé n'est utilisée dans le code.
