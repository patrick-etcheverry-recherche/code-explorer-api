◆ donnees

Information * Traitement::donnees
protected

La liste des informations jouant le rôle de donnée pour le traitement.

Cet attribut représente la liste des informations jouant le rôle de donnée pour le traitement.
Une information joue le rôle de donnée pour un traitement si sa valeur est nécessaire pour pouvoir exécuter le traitement.

◆ nom

string Traitement::nom
protected

Le nom du traitement.

Le nom du traitement peut prendre plusieurs formes. Il peut s'agir du nom d'une fonction d'une procédure ou d'une méthode mais le traitement d'un traitement peut également prendre la forme d'un commentaire.
Exemples :
calculerRayon() <– cas où le traitement est par exemple une fonction // Calculer le rayon <– cas où le traitement est nommé via un commentaire

◆ resultats

Information * Traitement::resultats
protected

La liste des informations produites par le traitement.

Cet attribut représente la liste des informations jouant le rôle de résultat pour le traitement.
Une information est un résultat pour un traitement si sa valeur est calculée ou modifiée par le traitement.
